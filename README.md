# Projecte Final MERN

Membres de l'equip: Adrià Garcia i Daniel Molina

## Descripció del projecte

Projecte de la optativa amb tematica de pelicules fet amb React + Typescript pel frontend i Javascript pel backend.

L’objectiu és convertir un projecte funcional de tematica lliure fet amb les tecnologies nombrades anteriorment.


## Arquitectura del projecte

L’aplicació està formada per:

- Frontend React + Typescript
- Backend Javascript
- Estils amb bootstrap

L'aplicacio permet:

- Crear pelicules
- Ediat pelicules
- Veure detalls de les pelicules
- Eliminar pelicules

## Requisits

Per executar el projecte cal tenir instal·lat:

- Node.js (si s’executa en local)


## Execució local

Instal·lar dependències dins de backend/ i frontend/projecte-api-pelicules/:
npm install
npm install bootstrap
npm install react-router-dom (a vegades no fa falta)

Executar:
npm run dev

Accedir a:
http://localhost:5173 pel frontend
http://localhost:3000 per comprovar si funciona la api
